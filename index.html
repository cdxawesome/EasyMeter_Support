Getting Started Guide

üöÄ Quick Start

  1. Open the app and tap "Create New Project"
  2. Enter your project name (e.g., "Apartment 101")
  3. Go to project details and add meters
  4. Start a reading session and record readings
  5. View results and export data as needed

‚ûï How to Add a Meter

  1. Go to Project Details ‚Üí Manage Meters
  2. Tap "Add Meter" button
  3. Choose meter type (Water, Electricity, Gas, Other)
  4. Enter meter name, location, and unit price
  5. Set multiplier if needed (default is 1)
  6. Save and your meter is ready

üìä How to Record Readings

  1. Go to Project Details ‚Üí New Reading Session
  2. Select each meter you want to read
  3. Enter the current meter value
  4. Review the batch details
  5. Tap "Complete" to save the reading session

üíæ How to Export Data

  1. Go to Settings ‚Üí Export Data
  2. Choose export format (Excel, JSON, etc.)
  3. Share or save the exported file
  4. To restore, use Import in the project list
Tip: Long press on readings to delete them, or use the batch details page

Frequently Asked Questions

Is my data secure and private?

‚ñº
Yes, your data is completely secure. All data is stored locally on your device using your device's secure storage. We never upload your readings, meter information, or any personal data to any server. The app works entirely offline except when you choose to use AI analysis features (which requires an internet connection to communicate with Google Gemini).

How do I back up my data?

‚ñº
You can export your project data in two ways:
1. Excel Export: Go to Project Details ‚Üí Export Reading Data (exports as .xlsx)
2. Project Backup: Go to Settings ‚Üí Export Data (exports project configuration as JSON)
Save these files to your computer or cloud storage for safe backup.

Can I use the app offline?

‚ñº
Yes! The app works completely offline for recording readings, managing projects and meters, and viewing historical data. The only feature that requires internet is the AI-powered data analysis (Gemini integration), which you can optionally use when you have an internet connection.

How do I import a previously exported project?

‚ñº
1. Go to the Project List (home screen)
2. Tap the "Import Project" button
3. Select the JSON backup file you saved earlier
4. The app will prompt you to confirm if it will create a new project or overwrite an existing one
5. Your data will be restored with all meters and readings

What is a "multiplier" and why do I need it?

‚ñº
Some meters display readings in units that need to be converted to standard units. For example:
- Many water meters multiply by 1 (display liters directly)
- Some electricity meters multiply by 40 (each unit = 40 kWh)
- The app uses this multiplier to calculate actual consumption: actual value = reading √ó multiplier

How is the cost calculated?

‚ñº
The cost is calculated as:
Cost = (Current Reading - Previous Reading) √ó Multiplier √ó Unit Price
For example: (100 - 95) √ó 40 √ó $0.15 = $30
You set the unit price when configuring each meter.

Can I delete a reading by mistake?

‚ñº
If you accidentally delete a reading batch, you can restore it from your backup:
1. Import your previously exported project file
2. Choose to overwrite your current project

We recommend regularly exporting your data as backup to prevent accidental data loss.

Does the app support multiple languages?

‚ñº
Yes! The app currently supports English and Simplified Chinese. You can change the language anytime in the app Settings. The app will remember your language preference.

What should I do if the app crashes?

‚ñº
If the app crashes:
1. Close and reopen the app
2. If the problem persists, uninstall and reinstall the app
3. Your data is safe in the device storage and will be restored
4. If you continue to have issues, please contact us with details

Can I edit meter information after adding it?

‚ñº
Yes! Go to Project Details ‚Üí Manage Meters, then tap on any meter to edit its name, location, type, multiplier, or unit price. Changes will apply to future calculations but won't affect past readings (they retain snapshots of the meter configuration at the time they were recorded).

Common Issues & Solutions

‚ùå Can't add a new meter

  1. Make sure you have created a project first
  2. Check that you have entered all required fields (name, type, price)
  3. Try closing and reopening the app
  4. Ensure your device has enough storage space

‚ùå Export button not working

  1. Make sure you have readings to export
  2. Check that your device has storage permission for downloads
  3. Try exporting a different project first
  4. Restart the app and try again

‚ùå Data not saving

  1. Ensure the app has local storage permissions
  2. Check that your device has adequate free storage
  3. Don't force-close the app while saving
  4. On iOS: Check Settings ‚Üí Mobile Meter Reader ‚Üí Local Network permissions
  5. On Android: Check Settings ‚Üí Apps ‚Üí Mobile Meter Reader ‚Üí Storage permissions

‚ùå AI Analysis not available

  1. Ensure you have an active internet connection
  2. Go to Settings ‚Üí Add API Key and enter your Google Gemini API key
  3. Check that your API key is valid and has sufficient quota
  4. Wait a moment and try again (API may be temporarily unavailable)
Important: If you encounter an issue not listed above, please contact support with details about:
  • Your device type and OS version
  • App version number
  • Steps to reproduce the issue
  • Screenshots if applicable
