What is a "multiplier" and why do I need it?

â–¼
Some meters display readings in units that need to be converted to standard units. For example:
- Many water meters multiply by 1 (display liters directly)
- Some electricity meters multiply by 40 (each unit = 40 kWh)
- The app uses this multiplier to calculate actual consumption: actual value = reading Ã— multiplier
